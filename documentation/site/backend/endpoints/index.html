API Endpoints

GET

  • /api/fish/ - Retrieve all fish species, and the details of each species.
    Expected Response:
    id: item.id | Unique identifier for the fish.
    species: item.species | Name of the species.
    bait: item.bait | List of bait associated with the fish
    food: item.food | List of food sources for the fish
    structure: item.structure | List of structures associated with the fish
    locations: item.locations | List of locations where the fish can be found
    trends: item.trends | List of trends related to the fish
    techniques: item.techniques | List of techniques for catching the fish
    tutorial_video: item.tutorial_video | URL for a tutorial video
